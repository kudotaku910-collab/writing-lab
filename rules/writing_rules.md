# 執筆ルール

このドキュメントは、本リポジトリにおける文章執筆の統一ルールを定めるものです。

## 1. 基本方針

-   専門的な医療情報を、一般の読者にも分かりやすく、正確に伝えることを目的とします。
-   読者に寄り添い、信頼感を与える丁寧なコミュニケーションを心がけます。

## 2. 文体・トーン

-   **敬体**: 全ての文章は「です・ます」調で統一します。
-   **語調**: 医師が患者に語りかけるような、穏やかで教育的なトーンを基本とします。
-   **断定表現**: 「～と言われています」「～と考えられます」といった表現を用い、断定的な物言いを避けます。ただし、予防や治療の重要性を訴える場面では、「～しましょう」「～が大切です」のように、行動を促す表現も使用します。

## 3. 文章構成

### 3.1. 全体構成

原則として、以下の順序で記事を構成します。

1.  **概要**: 記事全体のテーマと要点を簡潔に説明します。
2.  **目次**: 記事の主要なセクションを`■`を用いた箇条書きで示します。
3.  **詳細**: 各セクションの詳細な内容を記述します。
4.  **当院での取り組み（任意）**: 関連する診療案内や検査設備について紹介します。
5.  **まとめ・行動喚起**: 記事の要点をまとめ、読者へのメッセージ（受診勧奨など）で締めくくります。

### 3.2. 見出し

-   大見出しは `#` を使用します。（例: `# 概要`）
-   中見出しは `##` を使用します。（例: `## ラクナ梗塞`）
-   小見出しが必要な場合は `###` を使用します。

## 4. 表現・記法

### 4.1. 箇条書き

-   項目の列挙には `・` や `■` を使用します。
-   手順や番号付きリストには `①`、`②` のような丸数字を使用します。

### 4.2. 専門用語

-   専門用語や難しい漢字には、初出時にふりがなや簡単な説明を加えます。（例: `稀発（きはつ）反復性`）

### 4.3. 引用・参考資料

-   外部のガイドラインや資料から図や文章を引用する場合は、出典を明記します。（例: `高血圧治療ガイドライン2019より抜粋`）

### 4.4. 内部リンク

-   リポジトリ内の他の記事へ読者を誘導する場合、「〇〇について詳しく見る」「〇〇のご案内」という形式でリンクを設置します。

## 5. 読者への配慮

-   **読者への呼びかけ**: 「～だと思いませんか？」「～しましょう」といった形で、読者に行動や思考を促す表現を取り入れ、エンゲージメントを高めます。
-   **親近感**: 専門家としての信頼性を保ちつつも、読者が気軽に相談できるような親しみやすい雰囲気を作ります。
